# Buttonize Filter

The buttonize filter makes some links look like buttons if their link text ends in a `>` sign.

Markdown:

```md
[Normal Link](https://github.com/ripple/dactyl)

[Button Link >](http://github.com/ripple/dactyl)
```

Results:

[Normal Link](https://github.com/ripple/dactyl)

[Button Link >](http://github.com/ripple/dactyl)
